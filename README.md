# LMT2018
---
## 概览
老版本LMT已运行10年，长时间的维护周期使得LMT的维护成本越来越高，新版本LMT的迭代旨在优化老版本LMT存在的一些不足，优化软件架构，使得其更加便于维护和使用
## 运行环境与第三方框架
- **IDE**：Visual Studio 2015
- **.NET Framework:** 4.6.2
- **SNMP：** SnmpSharpNet
- **UML：** Enterprise Architect
